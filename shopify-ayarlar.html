Shopify bağlı değil

📋 Nasıl Yapılır?

Shopify API Bilgilerini Almak İçin:

  1. dev.shopify.com adresine git
  2. Apps → Create an app tıkla
  3. App name: SiparisAPI yaz, Create bas
  4. Sol menüden Settings tıkla
  5. Client ID ve Secret bilgilerini kopyala
  6. Aşağıya yapıştır ve Kaydet bas

🔑 Shopify API Bilgileri

Shopify mağazanızın adresi (örn: nexo.myshopify.com)
Settings sayfasındaki Client ID
Settings sayfasındaki Secret (göz ikonuna tıklayıp kopyala)
Token Al butonuna bas veya manuel olarak gir

⚙️ Entegrasyon Ayarları

Yeni sipariş geldiğinde otomatik olarak Shopify'da sipariş oluşturur
Shopify'daki stok değişikliklerini sisteme yansıt
