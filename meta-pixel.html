🔐 1. ADIM: Domain Doğrulama

Meta reklamları kullanmak için önce domain'inizi doğrulamanız gerekiyor.

📋 Nasıl Yapılır?

1. Meta Business → Domains sayfasına git
2. "Bir domain oluşturun" tıkla
3. Domain adını gir (örn: siteniz.com)
4. "Bir Meta etiketi ekleyin" seç
5. Verilen kodu aşağıya yapıştır

✅ Aşağıdaki kodu kopyala ve GitHub'daki index.html dosyasına ekle:

📍 Nereye Eklenecek?
1. GitHub'da index.html dosyasını aç
2. <head> etiketinden hemen sonra yapıştır
3. Commit et ve bekle (Vercel otomatik deploy edecek)
4. Meta'da "Domaini Doğrula" butonuna bas

🔧 2. ADIM: Meta Pixel ID Gir

📋 Pixel ID'yi Meta Events Manager'dan alın

ℹ️ Nasıl Kullanılır?

1. Meta Events Manager'a git
2. Pixel'ini seç veya yeni oluştur
3. Pixel ID'yi kopyala (sadece rakamlar)
4. Yukarıya yapıştır ve Kaydet'e bas
