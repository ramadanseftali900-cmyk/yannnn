⚙️ Detaylı Kurulum

1️⃣ Firebase Kurulumu

Adım 1: firebase.google.com adresine git

Adım 2: "Get Started" → Gmail ile giriş yap

Adım 3: "Add Project" → Proje adı gir

Adım 4: Google Analytics'i KAPAT

Adım 5: "Create Project" tıkla

Adım 6: Project Settings (⚙️) → "Your apps" → Web (</>)

Adım 7: App nickname gir → "Register app"

Adım 8: Açılan kodda şu bilgileri KOPYALA:

apiKey: "AIza..." authDomain: "proje.firebaseapp.com" databaseURL: "https://proje.firebaseio.com" projectId: "proje-adi" storageBucket: "proje.appspot.com" messagingSenderId: "123456" appId: "1:123:web:abc"

Adım 9: Sol menü → "Realtime Database" → "Create Database"

Adım 10: Location: "United States" → "Start in test mode"

Adım 11: "Enable" tıkla

Adım 12: "Rules" sekmesi → Şu kodu yapıştır:

{ "rules": { ".read": true, ".write": true } }

Adım 13: "Publish" tıkla

✅ Firebase Hazır! Şimdi bilgileri siteye gir:

1. domain-bagla.html dosyasını aç

2. "4. Firebase" sekmesine git

3. Kopyaladığın bilgileri yapıştır

4. "💾 KAYDET & TEST ET" tıkla

2️⃣ GitHub Kurulumu

Adım 1: github.com → "Sign up"

Adım 2: Email, şifre, kullanıcı adı belirle

Adım 3: Email'ini doğrula

Adım 4: "+" → "New repository"

Adım 5: Repository adı gir → "Public" seç

Adım 6: "Create repository"

Adım 7: "uploading an existing file" → Tüm dosyaları sürükle

Adım 8: "Commit changes" tıkla

3️⃣ Vercel Kurulumu

Adım 1: vercel.com → "Sign Up"

Adım 2: "Continue with GitHub"

Adım 3: "Add New..." → "Project"

Adım 4: Repository'ni seç → "Import"

Adım 5: "Deploy" tıkla (ayarları değiştirme)

Adım 6: Deploy tamamlanınca "Visit" tıkla

🎉 Tebrikler! Siteniz canlıda!

Adres: https://proje-adi.vercel.app

🐛 Sorun Giderme

❌ Firebase Bağlanamıyor

Sorun: "Firebase bağlantı hatası" mesajı alıyorum

Çözüm:

  • Firebase Console'da Realtime Database açık mı kontrol et
  • Database URL doğru mu kontrol et (sonunda .firebaseio.com olmalı)
  • Security rules "test mode" olmalı
  • Tarayıcı önbelleğini temizle (Ctrl+Shift+Delete)

❌ Görseller Yüklenmiyor

Sorun: Görseller yüklenmiyor veya görünmüyor

Çözüm:

  • Görsel boyutu 5MB'dan küçük olmalı
  • Görsel formatı JPG veya PNG olmalı
  • Tarayıcı localStorage'ı aktif olmalı
  • Gizli mod kullanmıyorsan emin ol

❌ Siparişler Kaydedilmiyor

Sorun: Sipariş veriyorum ama Firebase'de görünmüyor

Çözüm:

  • Firebase bağlantısını test et
  • Database rules "write: true" olmalı
  • Internet bağlantını kontrol et
  • Tarayıcı console'da hata var mı bak (F12)

❌ Admin Paneline Giremiyorum

Sorun: Şifre kabul edilmiyor

Çözüm:

  • Varsayılan şifre: hayat123
  • Tarayıcı önbelleğini temizle
  • localStorage'ı temizle (F12 → Application → Local Storage → Clear)
  • Sayfayı yenile (F5)

❌ WhatsApp Butonu Çalışmıyor

Sorun: WhatsApp butonu tıklanmıyor

Çözüm:

  • WhatsApp numarası doğru girilmiş mi kontrol et
  • Numara başında 0 olmamalı (5551234567)
  • "Butonu Göster/Gizle" toggle'ı açık olmalı
  • Mobilde WhatsApp yüklü olmalı
💡 İpucu: Sorun devam ederse tarayıcı console'unu aç (F12) ve hata mesajlarını kontrol et!
